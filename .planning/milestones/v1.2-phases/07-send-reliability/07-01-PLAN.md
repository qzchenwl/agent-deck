    **SEND-01: Harden Enter retry in sendWithRetryTarget (session_cmd.go)**

    In `sendWithRetryTarget`, change the waiting-without-active Enter nudge from every 3rd iteration (`retry%3 == 2`) to every iteration for the first 5 retries, then every 2nd iteration thereafter. This makes the retry more aggressive in the critical early window when bracketed paste timing failures are most likely to manifest. The change is in the `else` branch of the `sawActiveAfterSend` check (around line 1703):

    ```go
    // Old: if retry%3 == 2 { _ = target.SendEnter() }
    // New: more aggressive early nudge
    if retry < 5 || retry%2 == 0 {
        _ = target.SendEnter()
    }
    ```

    Also increase the ambiguous-state fallback Enter budget from `retry < 2` to `retry < 4` (line 1712), because some TUI frameworks take longer to process and reflect state.

    **SEND-01: Mirror the same fix in sendMessageWhenReady (instance.go)**

    Apply the identical changes to the inline retry loop in `sendMessageWhenReady` (lines 2035 and 2043):
    - Change `retry%3 == 2` to `retry < 5 || retry%2 == 0`
    - Change `retry < 2` to `retry < 4`

    **SEND-02: Add Codex readiness check to waitForAgentReady (session_cmd.go)**

    In `waitForAgentReady` (line 1723), after the existing Claude-specific composer check (line 1754) and before the final `time.Sleep(300ms)` return, add a Codex-specific readiness check:

    ```go
    if tool == "codex" {
        if rawContent, captureErr := tmuxSess.CapturePaneFresh(); captureErr == nil {
            content := tmux.StripANSI(rawContent)
            detector := tmux.PromptDetector("codex")
            if !detector.HasPrompt(content) {
                // Codex hasn't shown its prompt yet; keep polling
                continue
            }
        }
    }
    ```

    Place this AFTER the Claude check and BEFORE the `time.Sleep(300ms)` return. This ensures Codex sessions are not considered ready until the `codex>` prompt is actually visible in the pane.

    **SEND-02: Add Codex readiness check to sendMessageWhenReady (instance.go)**

    if i.Tool == "codex" {
