<objective>
Add SyncSessionIDsFromTmux to capture tool conversation IDs from the tmux environment before killing a session, and wire it into the stop path so resume operations can find the correct conversation.

Purpose: Resolves PLAT-02 (#311). The current handleSessionStop calls Kill() before reading tmux env vars. If PostStartSync timed out during start (tool started late on slow WSL2), ClaudeSessionID is never persisted, and resume falls back to creating a new conversation. The fix snapshots all four tool session IDs from tmux env into the Instance struct before destroying the tmux session.

Output: New `SyncSessionIDsFromTmux` method on Instance, modified `handleSessionStop` in session_cmd.go, new test file `internal/session/instance_platform_test.go`.
</objective>

    1. Add `SyncSessionIDsFromTmux()` method to `internal/session/instance.go`, placed immediately after `SyncSessionIDsToTmux()` (around line 2843). This is the reverse operation: reads session IDs FROM the tmux environment and updates the Instance struct fields.

    ```go
    // SyncSessionIDsFromTmux reads tool session IDs from the tmux environment
    // into the Instance struct. This is the reverse of SyncSessionIDsToTmux.
    // Used in the stop path to capture IDs that may not have been saved during
    // start (e.g., if PostStartSync timed out but the tool started late).
    // Only updates fields where the tmux env has a non-empty value; does not
    // blank existing IDs if the tmux env is missing the variable.
    func (i *Instance) SyncSessionIDsFromTmux() {
        if i.tmuxSession == nil || !i.tmuxSession.Exists() {
            return
        }

        if id, err := i.tmuxSession.GetEnvironment("CLAUDE_SESSION_ID"); err == nil && id != "" {
            i.ClaudeSessionID = id
            if i.ClaudeDetectedAt.IsZero() {
                i.ClaudeDetectedAt = time.Now()
            }
        }

        if id, err := i.tmuxSession.GetEnvironment("GEMINI_SESSION_ID"); err == nil && id != "" {
            i.GeminiSessionID = id
        }

        if id, err := i.tmuxSession.GetEnvironment("OPENCODE_SESSION_ID"); err == nil && id != "" {
            i.OpenCodeSessionID = id
        }

        if id, err := i.tmuxSession.GetEnvironment("CODEX_SESSION_ID"); err == nil && id != "" {
            i.CodexSessionID = id
        }
    }
    ```
